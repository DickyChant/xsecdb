Welcome to the Query page! To query a set of data against the database select a search term from the dropdown menu. If any of the search terms in the dropdown are unclear, a brief description of them can be found in the DB Tool User's Guide under the Insert section. After selecting a search term from the dropdown input the value into the Search Value field. The search value need only be a contiguous substring of the search term. For example, to query on the process name "DYJetsToLL" one could enter "jets", "dy", "toll", or any other contiguous substring. The query operation is NOT case sensitive. Do not use explicit wild cards such as "*" or "$", these are not supported. Thus one should NOT try to perform any query of the type: "*jets*ll".
